# Reserva

Site de reserva/aluguel de brinquedos.
Projeto todo constuido feito em Vue.js

## Tecnologias utilizadas:

1) Postgree
2) Vue.js
3) Node.js

## O Site foi hospedado na Digital Ocean.

Foi criado dois servidores

1) Postgree
2) Aplicação Vue.js (apache)
3) Aplicação noje.js (servidor rest)

## Setup do Projeto / Instalação
```
npm install
```

### Para iniciar o servidor
```
npm run serve
```

### Para Gerar o Build
```
npm run build
```
